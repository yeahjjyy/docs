## Endpoint Overview

Get the most recent tweets from a specific Twitter user using their username or user ID.

**Endpoint URL:** `GET https://api.twitterapi.io/twitter/user/tweets`

## Request Parameters

| Parameter | Type   | Required | Description                                           |
| --------- | ------ | -------- | ----------------------------------------------------- |
| userName  | string | Yes*     | Twitter username (without @)                          |
| userId    | string | Yes*     | Twitter user ID number                                |
| count     | number | No       | Number of tweets to return (default: 20, max: 100)    |
| includeReplies | boolean | No | Include replies in the result (default: false)        |
| includeRetweets | boolean | No | Include retweets in the result (default: false)      |

*At least one of userName or userId must be provided

## Code Examples

### cURL

```bash
curl --location 'https://api.twitterapi.io/twitter/user/tweets?userName=elonmusk&count=10' \
--header 'x-api-key: my_test_xxxxx'
```

### Python

```python
import requests

url = 'https://api.twitterapi.io/twitter/user/tweets'
params = {
    'userName': 'elonmusk',
    'count': 10,
    'includeReplies': False
}
headers = {'x-api-key': 'my_test_xxxxx'}
response = requests.get(url, params=params, headers=headers)
print(response.json())
```

### JavaScript

```javascript
const options = {
  method: 'GET',
  headers: {
    'x-api-key': 'my_test_xxxxx'
  }
};

fetch('https://api.twitterapi.io/twitter/user/tweets?userName=elonmusk&count=10', options)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```
