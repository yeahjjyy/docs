## Endpoint Overview

Get detailed information about a Twitter user using their username or user ID.

**Endpoint URL:** `GET https://api.twitterapi.io/twitter/user/info`

## Request Parameters

| Parameter | Type   | Required | Description                                      |
| --------- | ------ | -------- | ------------------------------------------------ |
| userName  | string | Yes*     | Twitter username (without @)                     |
| userId    | string | Yes*     | Twitter user ID number                           |

*At least one of userName or userId must be provided

## Response Format

```json
{
  "id": "44196397",
  "name": "Elon Musk",
  "username": "elonmusk",
  "bio": "Technoking of Tesla, Imperator of Mars",
  "followers_count": 169800000,
  "following_count": 150,
  "tweet_count": 34500,
  "listed_count": 120500,
  "created_at": "2009-06-02T20:12:29.000Z",
  "profile_image_url": "https://pbs.twimg.com/profile_images/1234567890/elon_musk_normal.jpg",
  "verified": true,
  "location": "Mars & Earth"
}
```

## Code Examples

### cURL

```bash
curl --location 'https://api.twitterapi.io/twitter/user/info?userName=elonmusk' \
--header 'x-api-key: my_test_xxxxx'
```

### Python

```python
import requests

url = 'https://api.twitterapi.io/twitter/user/info?userName=elonmusk'
headers = {'x-api-key': 'my_test_xxxxx'}
response = requests.get(url, headers=headers)
print(response.json())
```

### JavaScript

```javascript
const options = {
  method: 'GET',
  headers: {
    'x-api-key': 'my_test_xxxxx'
  }
};

fetch('https://api.twitterapi.io/twitter/user/info?userName=elonmusk', options)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

## Error Codes

| Status Code | Description                                        |
| ----------- | -------------------------------------------------- |
| 400         | Invalid request - missing required parameters      |
| 401         | Unauthorized - invalid or missing API key          |
| 404         | User not found                                     |
| 429         | Rate limit exceeded - too many requests            |
| 500         | Server error - please try again later              |
